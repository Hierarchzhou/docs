# 轻松上手 AETRIX 聊天

准备好和你的朋友或合作伙伴在 AETRIX 里聊聊创意了吗？

## 你需要准备：

1. 一个已经登录的 AETRIX 账户。
2. （可能需要）知道你想要聊天的对象的 AETRIX 用户名或昵称，或者他们也需要是你的平台好友/联系人（具体取决于平台设计）。

## 操作步骤：

<Tip>
**提示：** 建议参考界面截图或 GIF 动图，以便更直观地理解操作流程。
</Tip>

1. **找到聊天入口**：在 AETRIX 的主界面、导航栏或侧边栏，找到标有"**聊天**"、"**消息**"或类似图标/文字的入口，点击进入。

2. **开始新的对话**：
   - 你可能会看到一个联系人列表或好友列表。点击你想要聊天的对象即可开始对话。
   - 或者，可能有一个"**发起新聊天**"或"**+**"按钮。点击后，你需要搜索并选择你想要聊天的用户。

3. **发送消息**：进入聊天界面后，你会看到一个输入框。在输入框里输入你想说的话，然后点击"**发送**"按钮（通常是一个纸飞机图标）或按回车键。

4. **接收消息**：当有人给你发送消息时，你通常会在"聊天"入口看到未读消息提示（比如一个小红点或数字）。点击进入即可查看。

5. **管理聊天列表（可选）**：聊天主界面通常会显示你最近的对话列表。你可以方便地在不同对话间切换。

## 聊天功能小贴士：

  - **消息状态？** 有些聊天工具会显示消息是否已发送、已送达或已读。
