# Story Protocol 白皮书

## 点对点知识产权网络

### 故事基础

#### 摘要

我们提出了 Story，这是一个点对点的知识产权网络，为知识和创造力创造了一个可编程的市场。科学和创意资产通过可定制的使用参数注册在通用账本上。所有资产都配备了一个可组合的接口，任何软件应用程序或人工智能模型都可以使用该接口，从而允许知识产权在互联网上使用和货币化。一个网络范围的图协调所有知识产权资产，节点代表原子资产，边缘代表它们之间的法律和经济承诺。该网络通过由加密经济激励驱动的异步和去中心化验证服务评估每种资产的唯一性。参与该协议有助于世界知识和创造力唯一、开放且无需许可的存储库的发展。

## 1. 介绍

知识产权 (IP) 几乎完全依赖中心化中介机构来协调所有权和价值。从社交平台到制药公司，少数不负责任的实体负责生产和交换所有学科的情报，这增加了知识产权的交易成本。此外，传统系统无法再与人工智能的现实竞争，在人工智能中，创意可以大规模创建和分发，边际成本不断消失。个人发现他们的创意成果被抓取来训练模型，而没有署名和补偿，而人工智能公司本身也无法生成数亿份单独的许可协议。创意市场既没有可读性，也没有流动性。

我们需要的是一个开放且可编程的点对点知识产权交换市场。我们提出了 Story，一个点对点的知识产权网络。任何两个愿意的人都可以直接在 Story 上交易、扩展和货币化他们的想法，而不受寻租中介的干扰。所有者设定的可编程条款嵌入到每个知识产权资产中，因为它在 Story 网络上的任何软件应用程序之间货币化和交换，从而在互联网上为其所有者产生收益。Story 为增加人类智力输出所需的智能提供了通用存储库和可编程市场。

## 2. 架构概述

Story 是一个专门构建的去中心化区块链，由多核执行环境增强。它由一个主执行核心和多个高度定制的内核组成。主内核提供 EVM 等效性，从而能够快速采用生态系统中的现有应用程序。知识产权 (IP) 内核是专用内核之一，可有效地将知识产权注册作为原生资产类别处理，同时优化复杂 IP 关系图的作。该内核将智能转化为可编程的 IP 资产。

### 2.1 执行层

执行层作为网络的计算基础，处理所有交易和智能合约的执行。它由一个主内核和几个协同工作的专用内核提供支持。该层旨在为每个内核提供最大的灵活性和计算能力，同时保持网络的完整性和活跃性。

#### 知识产权核心

知识产权资产在 Story 上表示为多面图表，其中各种类型的节点（IP 资产、许可证等）通过代表经济和法律关系的边缘连接起来。这些 IP 图表跟踪原始资产和衍生资产的权利、许可和货币化。

创意证明 (PoC) 协议部署在 IP 内核上，是一种原生的协议，它提供了一个开放的 IP 存储库和一组模块，以便以无摩擦的方式与这些 IP 进行交互。PoC 既是 Story 中 IP 资产的标准化通用账本，也是围绕这些资产及其权利创建数字市场的一种方式。

### 2.2 存储层

该存储层位于共识层之上，并将存储 API 公开给执行层及其核心。它抽象出数据组织的复杂性，构建对数据的加密承诺以及数据库的选择。中央设计原则从 NAND 闪存转换层 (FTL) 中汲取灵感，该层将逻辑地址映射到物理存储块，以优化写入性能并延长设备使用寿命。

### 2.3 共识层

共识层通过协调所有网络参与者之间的区块形成来确保网络的完整性、安全性和稳定性。Story 的初始版本使用了名为 CometBFT 的现代高性能 Tendermint 实现。Tendermint 是一个拜占庭容错共识引擎，它提供定义明确的属性和自定义选项，以满足我们的设计需求。

## 3. 应用

### 3.1 通用的智能市场

Story 通过可编程 IP 实现点对点 IP 交易，消除了传统 IP 系统中的障碍。应用程序可以将制药公司或媒体机构的现有 IP 目录作为 IP RWA (真实世界资产) 整合到 Story 上。更令人兴奋的是，原生可编程的 IP 可以与 IPFi 新兴领域的 DeFi 应用程序组合，其中 IP 资产可以碎片化、用作抵押品、质押或以其他方式以经济高效的方式利用。

### 3.2 人工智能的基础层

区块链是 AI 交互的完美基础，因为它们提供了一种可编程的媒介，软件可以在其中做出硬承诺。由于代理以 IP 作为其原生输入进行训练，并生产 IP 作为其原生输出，因此 Story 为 AI 交易提供了一个基础结算层，用于训练 AI 模型和 AI 输出的货币化。

#### 智能链

在 AI 领域，知识产权资产包含各种形式的 IP，从数据集和基础 ML 模型到微调模型和模型调优包。在 Story 上注册整个智能链可以实现经济价值谱系，当收入在任何单个节点生成时，它会在整个图形中分配收入。

#### 代理商务

AI 正在从独立模型演变为可以感知、决定和采取行动以实现目标的自主代理网络。这代表着从将 AI 仅仅视为一种工具转变为将其视为一个生态系统，代理在其中协作并通过交互产生价值。

### 3.3 IP 令牌

Story 的原生代币 IP 协调了点对点知识产权系统中的所有价值流动。IP 代币充当网络的底层交换媒介。每当使用、交换或产生收入时，IP 通过透明的费用支付实现高效的交易处理和公平的资源分配。

## 结论

我们提出了一个不依赖中心化机构的点对点知识产权系统。Story 区块链提供了一个专门构建的网络，既可以创建知识产权资产的通用存储库，也可以创建用于交换这些资产的可编程市场。随着人工智能的加速，知识产权的数量和速度不断扩大，Story 的网络将成为新智能经济的支柱。比特币以数字黄金的形式作为所有商品资产的价值储存手段，而 Story 则以可编程知识产权的形式作为所有知识资产的价值储存手段。
